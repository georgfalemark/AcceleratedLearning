## Tips inför checkpoint

React behöver du **inte** bry dig om inför checkpointen (det ingår ej i uppgiften)

Öva på det vi gjort måndag-onsdag:
* Göra **GET**, **POST** med **fetch** i javascript
  * Kunna rita upp html ifrån vad servern svarar (tänk nyhetstabellen i NewsApp'en)
* Debuggning både i klienten och på servern (det kommer spara mycket tid)
* Översiktlig förståelse av Repository (se NewsApp'en)
* Controllers
